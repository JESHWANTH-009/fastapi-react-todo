# FastAPI + React ToDo App

This is a full-stack application combining **FastAPI** (Python backend) and **React** (frontend) to create a simple To-Do app. The backend provides APIs to create, read, update, and delete tasks, while the frontend allows users to interact with the tasks.

## Features
- Add new tasks
- Mark tasks as completed
- Delete tasks
- View all tasks

## Tech Stack
- **Backend**: FastAPI (Python)
- **Frontend**: React.js
- **Database**: SQLite
- **ORM**: SQLAlchemy

## Prerequisites
Ensure you have the following tools installed:
- Python >= 3.7
- Node.js >= 14.0
- npm (Node package manager)
